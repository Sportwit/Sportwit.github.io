    README¶ ↑

    Sportwit es una red social inspirada en Twitter para deportistas. El código fuente es libre y se puede consultar en Github puedes hacer clon y mejorar la aplicación según tus necesidades.

    Configuración básica:

    • Rails Version: 4.0.2

    • System dependencies:
      $ bundle install

    • Database creation:
      $ bundle exec rake db:create

    • Database initialization: $ bundle exec rake db:migrate

    • How to run the test suite:
      $ bundle exec rake db:populate
      $ bundle exec rake test:prepare

    • Deployment instructions (heroku)
      $ git push heroku
      $ heroku pg:reset DATABASE
      $ heroku run rake db:migrate
      $ heroku run rake db:populate

    En principio no tenemos pensado aceptar ningún pull request ya que el origen del proyecto es únicamente académico.
